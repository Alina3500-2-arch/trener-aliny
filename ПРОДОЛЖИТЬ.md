# Тренер Алины — где мы и что дальше

## Как продолжить в новом чате
Напиши: **«продолжаем Тренер Алины»** — я подхвачу всё из памяти.

## Готово (v7)
- 5 приёмов: Завтрак, Ланч, Обед, Полдник, Ужин.
- Голос: жмёшь 🎤 → запись сразу → распознаёт → уточняет граммы → можно править/удалять.
- Точные калории: база ~200 продуктов + «продукт + граммы» (калории сами). Нет в базе → считает ИИ.
- Кольцо калорий + плашка-прогноз («завтра на весах ≈ ±X г»).
- Онбординг с 3 вариантами похудения, профиль, напоминания.

## Дальше (по одному, аккуратно)
1. 📸 **Фото еды** — сфоткал прямо в приложении → ИИ распознал → уточнил граммы → калории. (следующее)
2. 🤖 **Авто-советы** — без кнопки; тренер сам пишет после еды + «другой вариант» блюда.
3. 💪 **Тренировки** — ИИ составляет программу, ты по ней занимаешься.

## Как обновить приложение на телефоне
Закрыть приложение → Expo Go → вкладка **preview** → верхняя запись. (Вход в Expo Go под `aline456`.)

## Не забыть
Пересоздать/удалить ключ Groq и токен Expo на сайтах (они светились в чате).
