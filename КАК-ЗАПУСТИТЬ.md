# Тренер Алины — как открыть на телефоне

## Что это
Персональный ИИ-тренер: дневник калорий, вес, цель со стратегией, тренировки, история.
Данные хранятся прямо на телефоне.

## Открыть на телефоне (без App Store)
1. Установи на телефон приложение **Expo Go** (App Store / Google Play).
2. На компьютере в папке проекта выполни:
   ```
   cd C:\Users\Flower\trener-aliny
   npx expo start
   ```
3. Появится **QR-код**. Отсканируй его:
   - iPhone — камерой;
   - Android — прямо из приложения Expo Go.
4. Приложение откроется на телефоне. Компьютер и телефон должны быть в **одной Wi-Fi сети**.

## Что уже работает (слой 1)
- Вкладка «Сегодня»: кольцо калорий, БЖУ, приёмы пищи, советы тренера.
- «Вес»: запись взвешиваний + график.
- «Цель»: постановка цели → расчёт стратегии (дефицит, БЖУ, темп).
- «Тренировки»: программа 3 дня + запись выполнения.
- «История»: всё по дням.
